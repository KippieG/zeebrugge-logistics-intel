Method

How The Signals Were Scored

The credibility comes from labeling what is observed, what is inferred and what still needs field validation.

Observed

A public source names the system, platform, product, or vendor directly.

Inferred

The stack is operationally likely from terminal, customs, warehouse, RoRo, rail, or forwarding activity, but no vendor is named.

Unknown

No reliable public signal was found. These entries need interviews, job-posting checks, vendor cases, or direct outreach.

1

Register sources in CSV with source type and notes.

2

Scrape text extracts with a dependency-free Python script.

3

Map companies to services, stack signals, confidence, and source IDs.

4

Validate structure locally and in GitHub Actions.
